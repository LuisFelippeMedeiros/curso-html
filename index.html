O que é HTML

HTML (HyperText Markup Language) é a linguagem padrão usada para criar e estruturar páginas web. Ele é composto por uma série de elementos, que são representados por tags. Essas tags informam ao navegador web como exibir o conteúdo da página. Aqui estão alguns conceitos básicos de HTML:!DOCTYPE html: Declaração do tipo de documento que informa ao navegador que o documento é HTML5. html: A raiz do documento HTML. head: Contém meta-informações sobre o documento, como o título. title: Define o título da página, que é exibido na aba do navegador. body: Contém o conteúdo visível da página, como textos, imagens, links, etc. h1 a h6: Títulos de diferentes níveis. p: Parágrafo de texto.

Tag P

A tag p em HTML é usada para definir um parágrafo de texto. O conteúdo dentro das tags p é exibido como um bloco de texto separado, com um espaço em branco acima e abaixo, o que ajuda a organizar e separar o conteúdo em uma página web.

Características da Tag p Bloco de Texto: Cada p cria um bloco de texto separado. Espaçamento: Navegadores geralmente adicionam um espaço em branco (margem) antes e depois de cada parágrafo, para melhorar a legibilidade. Conteúdo: Pode conter texto, links, imagens e outras tags inline.

DIV

Uma div é um elemento HTML que representa uma divisão ou seção em um documento HTML. A tag div é usada para agrupar blocos de conteúdo e aplicar estilos ou scripts a eles de maneira conjunta. É um dos elementos mais comuns e versáteis em HTML e é frequentemente usado em conjunto com CSS e JavaScript para criar layouts complexos e interativos em páginas web.

Listas ordenadas e Não ordenadas

Em HTML, listas ordenadas e não ordenadas são duas formas de agrupar itens de forma organizada. Elas são definidas usando as tags ol (lista ordenada) e ul (lista não ordenada).

Lista Ordenada (ol) Uma lista ordenada é uma lista de itens em que a ordem dos itens é significativa. Cada item é numerado automaticamente pelo navegador. A tag ol é usada para criar listas ordenadas e a tag li (list item) é usada para definir cada item da lista.

Lista Não Ordenada (ul) Uma lista não ordenada é uma lista de itens em que a ordem dos itens não é significativa. Os itens são geralmente exibidos com marcadores (bolinhas) pelo navegador. A tag ul é usada para criar listas não ordenadas e a tag li (list item) é usada para definir cada item da lista.

Links

A tag link em HTML é usada para definir a relação entre o documento atual e um recurso externo. Ela é comumente usada dentro da seção head do documento para linkar folhas de estilo externas (CSS), ícones de sites (favicon), e outros recursos necessários para a página web. A tag link é uma tag vazia, o que significa que ela não possui uma tag de fechamento.
